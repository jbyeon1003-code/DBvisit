사용 방법
1. ims.dbhitek.com 접속
2. F12 → Console 탭 열기
3. 복사한 스크립트 붙여넣기 (Ctrl+V) → Enter
4. 방문신청 버튼 클릭 → 동의 → 폼 입력까지 자동 진행됩니다.
5. 입력 완료 후 내용 확인하고 직접 제출하세요.
